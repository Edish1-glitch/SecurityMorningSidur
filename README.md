## 🔍 על הפרויקט

**סידור שמירת בוקר** הוא אפליקציית ווב לניהול וייצור סידורי שמירה למשמרת בוקר (07:00–15:00).

המערכת מייצרת אוטומטית סידור מאוזן ותקין תוך שמירה על כל חוקי השמירה — כולל חוקי רציפות, הגבלות CICO, ואיזון הפסקות בין השומרים.

```
┌──────────────┬──────────────────────────────────────────┐
│   עמדה       │   תיאור                                  │
```

---

## 👮 רמות השומרים

- **אחמ"ש** — מפקד המשמרת. מוגבל ב-CICO רק כשנדרש, ומחויב לכל העמדות הייצוגיות.
- **חזק** — שומר בכיר. מוגבל ל-CICO אחד בלבד אם יש אחמ"ש.
- **רגיל** — שומר סטנדרטי עם עד 2 CICO במשמרת.
- **שער (כפולה)** — שומר שער. בשעה האחרונה חייב להיות ב-CICO (כפולה).

---

## 📏 חוקי הסידור

המערכת אוכפת את כל הכללים הבאים באופן אוטומטי:

- ✔️ אחרי CICO — חובה שעת הפסקה
- ✔️ לא יותר מ-3 שעות עמדה פעילה ברצף
- ✔️ Lenel ו-Bosh חייבות להיות מאוישות בכל שעה
- ✔️ מלשינון לא יכולה להיות רצופה (שעתיים ברצף)
- ✔️ הפסקות לא יכולות להיות רצופות לאותו שומר
- ✔️ איזון הפסקות — פער מקסימלי של שעה אחת בין השומרים
- ✔️ מקסימום 3 שעות מנוחה (הפסקה + מלשינון) לשומר
- ✔️ כל שומר רגיל מבצע את כל סוגי העמדות לפחות פעם אחת
